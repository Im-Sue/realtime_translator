# 04_模块规格

这个目录存放业务模块级规格说明，强调“边界、职责、接口、状态”，不是简单按代码目录拆分。

当前采用业务能力拆分，建议每个模块一篇文档，至少写清楚：

- 模块职责是什么
- 对外暴露什么接口
- 依赖哪些上下游
- 当前实现状态和限制
