## 当前同步状态

本目录已按 2026-04-04 分支现状同步，重点反映以下变化：

- 桌面端已经成为主入口，CLI 模式退居兼容路径。
- `CH1` 默认输出链路已切到 `PCM 48kHz` 直出，`ogg_opus` 作为兼容回退。
- `CH2` 字幕不再按旧的文本拼接描述，而是基于 `start / streaming / end` 生命周期事件驱动。
- 配置管理新增深度补齐逻辑，旧配置加载后会自动补足新增字段。

## 使用原则

- 写“当前已经实现了什么”，不要写成理想化架构图。
- 写“谁调用谁、谁依赖谁”，不要只写代码文件名。
- 写“已知限制和真实耦合点”，方便后续重构时定位边界。
